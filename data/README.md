# Dataset Documentation
This project uses a 3-phase induction motor current dataset.
- **Structure:**
  - `Healthy/`: Baseline motor current data.
  - `BRB_xxx/`: Broken Rotor Bar fault data.
  - `BFI_xxx/`: Bearing Fault Inner Race data.
  - `BFO_xxx/`: Bearing Fault Outer Race data.
- **Format:** CSV files (Time-series data, 3-columns for Phase A, B, C).
